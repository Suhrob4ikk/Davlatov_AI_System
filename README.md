# Предсказание уровня преступности по округам США

Проект по машинному обучению в рамках курса «Системы ИИ».  
Датасет: [County Health Rankings](https://www.countyhealthrankings.org/)

---

## О проекте

В датасете собраны данные о здоровье и качестве жизни по **3000+ округам США** за период 2003–2014 гг.  
Каждая строка — один показатель (безработица, преступность, ожирение и др.) для одного округа в один временной период.

**Задача:** предсказать уровень насильственной преступности (`Violent crime rate`) —  
число насильственных преступлений на 100 000 жителей.

**Тип задачи:** регрессия (предсказание числового значения).

## Методология

### 1. Предобработка данных
- Фильтрация по показателю `Violent crime rate`
- Удаление строк с пропусками только по используемым колонкам (`dropna(subset=...)`)
- Исключение колонок с 100% пропусков (`Confidence Interval Lower/Upper Bound`)

### 2. Разбивка данных
- Train: 75% (11 067 строк)
- Validation: 25% (3 689 строк)
- `random_state=42` для воспроизводимости

### 3. Модели
- **DecisionTreeRegressor** — дерево решений без ограничений
- **DecisionTreeRegressor** с подбором `max_leaf_nodes` (5, 50, 100, 500, 1000, 5000, 10000)
- **RandomForestRegressor** — ансамбль из 100 деревьев

### 4. Метрика
**MAE (Mean Absolute Error)** — средняя абсолютная ошибка.  
Показывает на сколько единиц (преступлений на 100 000 жителей) модель ошибается в среднем.

---

## Результаты

| Модель | MAE | Ошибка от среднего |
|--------|-----|-------------------|
| Дерево решений (без ограничений) | ~11.6 | ~4.2% |
| Дерево решений (оптимум листьев) | ~11.8 | ~4.3% |
| **Случайный лес (100 деревьев)** | **~6.1** | **~2.2%** |

**Лучшая модель:** случайный лес.  
**Улучшение** по сравнению с одним деревом: ~47%.
